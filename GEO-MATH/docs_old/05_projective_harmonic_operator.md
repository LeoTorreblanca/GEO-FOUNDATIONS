# Projective Harmonic Operator

## Purpose

The previous documents established conservative decomposition and quotient geometry.

The canonical operators quantify participation but do not yet describe how complementary sectors evolve.

The present document derives the unique harmonic representation compatible with conservative complementarity.

---

# Starting Point

Let

[
A(\lambda)
]

and

[
B(\lambda)
]

be two complementary participation functions depending continuously on an internal parameter

[
\lambda.
]

By conservative decomposition,

[
A(\lambda)+B(\lambda)=1.
]

Both quantities must remain non-negative.

Therefore

[
0\le A\le1,
\qquad
0\le B\le1.
]

---

# Differential Conservation

Differentiating,

[
A'(\lambda)+B'(\lambda)=0.
]

Thus every admissible variation in one sector is exactly compensated by the opposite variation in the complementary sector.

The evolution is tangent to the conservative manifold.

---

# Positivity Requirement

Since both sectors represent participation,

negative values are forbidden.

The representation must satisfy simultaneously:

* conservation;
* continuity;
* positivity;
* boundedness.

---

# Harmonic Representation

The simplest continuously differentiable functions satisfying

[
A+B=1
]

while remaining positive and bounded are

[
A(\phi)=\cos^2(\phi),
]

[
B(\phi)=\sin^2(\phi).
]

Immediately,

[
\cos^2(\phi)+\sin^2(\phi)=1.
]

Thus harmonic projection realizes conservative complementarity exactly.

---

# Uniqueness

Any admissible representation preserving:

* positivity;
* normalization;
* smoothness;
* symmetry;

may be written as

[
A=f^2,
\qquad
B=g^2,
]

with

[
f^2+g^2=1.
]

The normalized Euclidean solution is precisely

[
f=\cos,
\qquad
g=\sin.
]

Therefore

[
\boxed{
A=\cos^2,
\qquad
B=\sin^2.
}
]

The harmonic representation is the canonical realization of conservative complementarity.

---

# Projective Interpretation

The parameter

[
\phi
]

is not merely an angle.

It parametrizes redistribution along the conservative manifold.

Projection corresponds to movement over this harmonic orbit.

Observable participation and complementary participation become orthogonal projections of the same primitive organization.

---

# Canonical Harmonic Operator

Define

[
\boxed{
H(\phi)=\cos^2(\phi).
}
]

Its complementary operator is

[
\boxed{
H_\perp(\phi)=\sin^2(\phi).
}
]

Together,

[
H+H_\perp=1.
]

This identity is the functional realization of the algebraic conservation law.

---
